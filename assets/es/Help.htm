ACTUALIZACIÓN DE SOFTWARE

Puedes comprobar si existe una nueva versión de software para tu equipo.

Existen dos modos de hacerlo:

   1) Localmente mediante archivo (debes disponer del archivo de actualización dentro de la memoria interna)

   2) Remotamente desde servidor (comprobación automática, debes tener conexión a Internet)

Importante: Asegúrate de tener batería suficiente (por lo menos un 50%) para proceder con la actualización. En caso de no ser así, conecta el equipo a la red antes de comenzar el proceso. Si ocurriera algún error en el proceso, trata de resetear el equipo, o llama al Servicio Técnico para más información.
